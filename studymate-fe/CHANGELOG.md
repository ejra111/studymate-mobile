## [1.1.0] - 2024-05-29

### Added
- **AI Coach: Tombol clear chat untuk menghapus semua pesan
- **AI Coach: Tombol copy jawaban untuk menyalin jawaban AI
- **Skeleton Loaders**: Animasi shimmer selama data dimuat di semua halaman
- **Error States**: Tampilan error dengan tombol "Coba Lagi"
- **Light Theme**: Dukungan tema terang dan tombol toggle tema
- **Font Size Adjustment**: Pengaturan ukuran teks di Profile Tab
- **Retry Mechanism**: Retry otomatis 3x untuk load data
- **Unit Tests**: Test untuk AppController
- **Widget Tests**: Test untuk SkeletonLoader dan ErrorState
- **Documentation**: Update README.md dan CHANGELOG.md

### Fixed
- **State Management**: Rollback state ketika API call gagal
